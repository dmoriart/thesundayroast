# Press Assets

This directory contains high-resolution assets for press and promotional use.

## Files to Add

- `band-photo-cmyk.jpg` - High-res band photo in CMYK color space (for print)
- `band-photo-rgb.jpg` - High-res band photo in RGB color space (for web/digital)
- `tech-rider.pdf` - Technical rider for live performances
- `stage-plot.pdf` - Stage plot diagram

## Usage

These files are referenced in the EPK (Electronic Press Kit) at `/press` and can be downloaded by promoters, venues, and press contacts.
