## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Customize for a New Artist

1. Update `band_data.json` with your band's information
2. Replace `public/images/hero-image.jpg` with your band's photo
3. Deploy!

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- React

## Deployment

Deploy easily to:
- **Vercel** (recommended for Next.js)
- **Netlify** (includes `netlify.toml` configuration)
